# Адаптивный тренажёр ПДД с ИИ

Интерактивное приложение для подготовки к экзамену ПДД. ИИ анализирует ответы и подбирает вопросы из ваших слабых тем.

## Как работает ИИ

1. Отслеживает процент правильных ответов по каждой теме
2. Находит тему с наихудшим результатом (минимум 2 ответа)
3. Задаёт следующий вопрос из этой темы
4. После тренировки даёт персональные рекомендации

## Содержание

5 тем: Знаки, Скорость, Разметка, Обгон, Пешеходы
12 вопросов с полным разбором каждого ответа

## Возможности

- Теория ПДД по каждому вопросу
- Описание опасностей и штрафов
- Реальные дорожные ситуации
- Полная статистика после тренировки

## Запуск

Скачайте файлы (index.html, style.css, script.js) в одну папку и откройте index.html в браузере.
